About Us

We are a group of fans really passionate about Star Wars. We decided to build this online encyclopedia using the Star Wars API which is the world's first quantified and programmatically-formatted set of Star Wars data. After hours of watching films we present to you all the People, Films, Species, Starships, Vehicles and Planets from Star Wars. We've formatted this data in JSON and exposed it to you in a RESTish implementation that allows you to programmatically collect and measure the data.
