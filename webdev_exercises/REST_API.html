The Star Wars Character Enciclopedia uses asynchronous HTTP (Ajax) request, it loads data from SWAPI (Star Wars Api) using a HTTP GET request. The Star Wars API is the world's first quantified and programmatically-formatted set of Star Wars data. It contains all the People, Films, Species, Starships, Vehicles and Planets from Star Wars. The data is formatted in JSON and exposed in a RESTish implementation that allows you to programmatically collect and measure the data.

This app also uses JavaScript promises. Promises provide a clean, flexible way to chain multiple async operations together without having to nest function calls inside callbacks. Promises where used to handle errors, and to display the HTTP status codes. If there is a connection error a alert message will appear with the status number.
